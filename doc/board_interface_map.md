# MES50HP 已确认接口总表

本文档只整理当前已经确认、后续写约束和代码会直接用到的接口。

## 1. 时钟

| 信号 | 说明 | FPGA 管脚 | 备注 |
| --- | --- | --- | --- |
| FPGA_GCLK_50M | 50MHz 有源系统时钟 | P20 | FPGA 主系统时钟源 |
| FPGA_GCLK_27M | 27MHz 有源参考时钟 | K21 | 当前项目暂不优先使用 |

## 2. 串口调试

| 信号 | 说明 | FPGA 管脚 | 备注 |
| --- | --- | --- | --- |
| UART0_TX | UART 数据输出 | R9 | PC 串口接收 |
| UART0_RX | UART 数据输入 | R8 | PC 串口发送 |

## 3. 用户按键

| 信号 | 说明 | FPGA 管脚 | 有效电平 | 备注 |
| --- | --- | --- | --- | --- |
| RESET / REST | 复位按键 | 待确认是否与 RSN 同一逻辑 | 低有效 | 文档中出现 REST，建议统一写 RESET |
| KEY1 | 用户按键 | K18 | 低有效 | 按下为低 |
| KEY2 | 用户按键 | L15 | 低有效 | 按下为低 |
| KEY3 | 用户按键 | J17 | 低有效 | 按下为低 |
| KEY4 | 用户按键 | K16 | 低有效 | 按下为低 |
| KEY5 | 用户按键 | J16 | 低有效 | 按下为低 |
| KEY6 | 用户按键 | J19 | 低有效 | 按下为低 |
| KEY7 | 用户按键 | H20 | 低有效 | 按下为低 |
| KEY8 | 用户按键 | H17 | 低有效 | 按下为低 |

## 4. 当前项目可直接使用的最小集合

如果先做最小系统和调试链路，下面这些就够用：
- FPGA_GCLK_50M -> P20
- UART0_TX -> R9
- UART0_RX -> R8
- RESET / KEY 之一作为复位输入

## 5. 还需要补充的接口

如果要继续完成板卡到 PC 的真实联调，还需要这些信息：
- 1000M PHY 的完整管脚分布
- PHY 接口模式（RGMII / GMII / 其他）
- PHY 复位脚
- MDC / MDIO
- PHY 参考时钟相关信号
- PHY 所在 IO Bank 和电压标准

## 6. 使用建议

1. 先用 50MHz + UART + 复位 做最小系统。
2. 再补 PHY 管脚，做网口协议联调。
3. 最后再接摄像头和后续图像链路。
