# 标注格式

## 1. CSV 格式建议
如果你暂时不想用专业标注工具，可以直接发一个 CSV 给我。

字段建议如下：
- `filename`：图片文件名
- `plate_text`：车牌文本
- `x`：左上角 x
- `y`：左上角 y
- `w`：宽
- `h`：高
- `notes`：可选备注，如夜晚、模糊、遮挡

## 2. 适用场景
- 车牌裁剪图：只需要 `filename` + `plate_text`
- 场景整图：需要 `filename` + `plate_text` + `x,y,w,h`

## 3. 你也可以直接发我什么
- 图片压缩包
- 图片文件夹
- CSV 或 Excel 标注表
- 只有图片也行，我可以先帮你整理成待标注清单
