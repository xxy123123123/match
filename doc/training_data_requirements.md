# 训练数据要求

## 1. 训练目标
当前项目的训练优先级建议如下：
1. 先做车牌识别字符训练。
2. 再做车牌检测训练。
3. 最后做场景适配和复杂光照增强。

## 2. 你提供图片时的最小要求
如果你现在就要开始给图片，请至少满足下面之一：

### 方案 A：车牌裁剪图 + 文本
- 图片中只保留单个车牌。
- 每张图对应一个车牌文本。
- 最适合先做 OCR 训练。

### 方案 B：整车/场景图 + 框 + 文本
- 图片中包含车辆场景。
- 同时提供车牌位置框。
- 同时提供车牌文本。
- 最适合做检测和识别联合训练。

## 3. 文件组织建议
- 原始图片放在 `dataset/plate_train/raw/`
- 已标注图片放在 `dataset/plate_train/labeled/images/`
- 检测标注放在 `dataset/plate_train/labeled/annotations/`
- 文本标注放在 `dataset/plate_train/labeled/ocr/`

## 4. 我最需要的字段
对每张图片，最好给出：
- 文件名
- 车牌文本
- 是否为裁剪图
- 是否有检测框
- 备注，如夜晚、逆光、模糊、遮挡

## 5. 后续训练前我会做什么
1. 检查图片尺寸和清晰度。
2. 整理训练/验证/测试集。
3. 统一标注格式。
4. 再决定是先训练检测器还是识别器。
